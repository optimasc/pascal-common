function isvalidcountrycode(s: shortstring): boolean;

Verifies if the 2 letter country code is valid

This routine checks if the two letter country code is valid (as defined in ISO3166-1). The country code is not case sensitive.

parameters
s
The three digit country code
returns

TRUE if the country code is valid, otherwise returns FALSE
