indicates if the times are in UTC format, this is always true, unless the filesystem does not support this information.

dev: array[0..127] of char;

Device where this file resides, this value is represented as an hexadecimal null terminated string and is operating system dependent.

ino: array[0..127] of char;

Unique file serial number, this may change from one boot to the next.
