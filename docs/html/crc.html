function UpdateCrc32(InitCrc:longword; b: byte):longword;

Routine to get the CRC-32 value. Normally to be compatible with the ISO 3309 standard, the first call to this routine should set InitCRC to $FFFFFFFF, and the final result of the CRC-32 should be XOR'ed with $FFFFFFFF.

parameters
InitCRC
The value of the previous CRC
b
The data byte to get the CRC-32 of
returns

The updated CRC-32 value
